货物 ratio： 价值 val + 直线距离 distance

泊位 ratio：装卸速度 velocity + time (轮船到虚拟点的时间) + 直线距离 distance

$goodsRatio=Val / distance $

$berthRatio = 1000 * Velocity / (time + distance) $
